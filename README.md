
	- Python 3.10+ is required.

	  ```powershell

	  .venv\Scripts\Activate
	  ```

	```powershell

	```

	- Open the `notebooks/` folder in VS Code or Jupyter.
	- Start with `1.PromptEngineering.ipynb` and follow the sequence, or jump to any topic of interest.
